MIND-CTRL

MIND-CTRL is an interactive, real-time video project controlled via a Brain Computer Interface (BCI). It uses machine learning to classify EEG sensor data in real-time.
The audio-visual component of MIND-CTRL transforms image to particle system. The movement of the particle system is a reflection of the user's mental state, via the BCI.
By maintaining a meditative state, the user is able to control the system, causing the particles to coalesce and reveal the original image.
A new image is then automatically loaded and the interaction cycle starts again.
